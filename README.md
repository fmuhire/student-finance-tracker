This is a simple web application that helps students track money they spend and manage their
budget.

Users can add records such as food, transport, and tuition.

The application shows:
- Total spending
- Number of transactions
- Top spending category
- Budget status (over or within budget)

Features:

- Add financial records
- Search records
- Sort records by newest, oldest, highest, and lowest
- Dashboard with spending summary
- Budget tracking system
- Data saved automatically using local storage
- Export data as JSON file
- Import data from JSON file
- Simple responsive design

Technologies Used:

- HTML
- CSS
- JavaScript
- Local Storage API
- FileReader API

How to use:

1. Open index.html in a browser
2. Add a record with description, amount, category, and date
3. View updates in the dashboard
4. Use search and sort features
5. Export or import data when needed
